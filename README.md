# ⛽ Calculadora Etanol x Gasolina

# Aplicacao Web desenvolvida com HTML, CSS e JavaScript para auxiliar na escolha entre abastecer com Etanol ou Gasolina com a utlização da regra dos 70%.

## 🚀 Funcionalidades

- Informar o preco do Etanol
- Informar o preco da Gasolina
- Comparar os valores utilizando a regra dos 70%
- Exibir qual combustivel e mais vantajoso
- Inverter os valores dos campos
- Instalacao como PWA (Progressive Web App)
- Funciona em dispositivos moveis e computadores

## Regra Utilizada

Se:

Etanol <= 70% do valor da Gasolina

Entao:

Abasteca com Etanol

Caso contrario:

Abasteca com Gasolina

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PWA (Progressive Web App)
